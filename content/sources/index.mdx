# Source registry

The registry keeps external material and derived artifacts distinguishable. A source link does not imply endorsement; it records where evidence or context originated.

## Active datasets and artifacts

### NARIT GHOSTS halo catalogs

The working dataset repository contains original analysis figures and catalog-derived outputs used by the M81/M82 figure archive.

[Open the dataset on Hugging Face ↗](https://huggingface.co/datasets/appleboiy/narit-ghosts-halo-catalogs)

### M81 / M82 Figure Archive

The visual evidence layer preserves full figures, analysis-stage labels, update metadata, and links to original-resolution sources.

[Browse the archive ↗](https://archive.chaipat.cc)

### chaipat.cc project registry

The central identity node records project descriptions, technical specifications, and links across the wider research ecosystem.

[Open the project registry ↗](https://chaipat.cc)
