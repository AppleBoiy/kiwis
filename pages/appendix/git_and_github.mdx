## Installing Git and Adding SSH Key for GitHub (macOS)

<Steps>

### Step 1: Install Git

1. Open `Terminal`.

   For more information please see:

   <Card icon={<TerminalIcon />} title="How to open terminal in macOS" href="/faq/how_to_open_terminal"></Card>

2. Check if Git is already installed by running the following command:

   ```sh copy filename="console"
   $ git --versio
   > git version 2.33.1
   ```

   <Callout type="info">
     If you see the Git version information, skip to [Step 2: Generate SSH
     key](#step-2-generate-ssh-key). Otherwise, continue to the next step.
   </Callout>

3. Install Homebrew

   <Card icon={<BrewIcon />} title="How to Install Homebrew on macOS" href="/docs/homebrew"></Card>
   <Callout >
   If you already have `Homebrew` installed on your macOS, you can skip this step.
   </Callout >

4. Once Homebrew is installed, run the following command to install Git:

   ```sh copy filename="sh"
   brew install git
   ```

### Step 2: Generate SSH Key

1. Run the following command to generate a new SSH key:

   ```sh copy filename="sh"
   ssh-keygen -t ed25519 -C sample@mail.com
   ```

   <Callout>
     Replace `sample@mail.com` with your own email address associated with your
     GitHub account.
   </Callout>

2. You will be prompted to enter a file in which to save the key. Press <kbd>Enter</kbd> to accept the default location.

   <img src="https://raw.githubusercontent.com/CSCMU-65s/cs-wiki101/main/img/github/ssh-keygen.jpeg" />

### Step 3: Add SSH Key to GitHub

1. Open a web browser and go to [GitHub](https://github.com/).
2. Click on your profile picture in the top-right corner

   <img src="https://raw.githubusercontent.com/CSCMU-65s/cs-wiki101/main/img/github/github-corner-profile.jpeg" />

   then select `Settings` from the dropdown menu.

   <img src="https://raw.githubusercontent.com/CSCMU-65s/cs-wiki101/main/img/github/github-settings.jpeg" />

3. In the left sidebar, click on `SSH and GPG keys` Click on the `New SSH key`.

   <img src="https://raw.githubusercontent.com/CSCMU-65s/cs-wiki101/main/img/github/add-ssh-key.jpeg" />

4. Switch to the Terminal application and run the following command to copy your public SSH key to the clipboard:

   ```sh copy filename="sh"
   pbcopy < ~/.ssh/id_ed25519.pub
   ```

   **OR** you can run the following command to print the key to the Terminal and copy it manually:

   ```sh copy filename="sh"
   cat ~/.ssh/id_ed25519.pub
   ```

   You should see a message like:

   ```sh filename="console"
   $ cat ~/.ssh/id_ed25519.pub
   > ssh-ed25519 your_key...blablabla your.mail@example.com
   ```

5. Return to the browser and paste the copied key into the `Key` field on the GitHub website.
6. Click the <kbd>Add SSH key</kbd> button to save the key.

   <img
     src="https://raw.githubusercontent.com/CSCMU-65s/cs-wiki101/main/img/github/add-ssh-key-button.jpeg"
     alt="add-ssh-key-button"
   />

   You have successfully added your SSH key to your GitHub account.

### Additional: Test SSH Connection

1. In the Terminal application, run the following command to test the SSH connection with GitHub:

   ```sh copy filename="sh"
   ssh -T git@github.com
   ```

2. You may see a warning message about the authenticity of the host. Type "yes" to continue.
3. If everything is set up correctly, you should see a message like:

   <img
     src="https://raw.githubusercontent.com/CSCMU-65s/cs-wiki101/main/img/github/check-connect-ssh.jpeg"
     alt="check-connect-ssh"
   />

   <Callout type="info">
     In this case `AppleBoiy` is my GitHub username. You should see your own
     username instead.
   </Callout>

</Steps>
