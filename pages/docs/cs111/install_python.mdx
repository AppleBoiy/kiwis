    ## Why Use the Python Installer?

    1. **User-friendly Interface**: The Python installer provides a graphical user interface (GUI) that guides you through the installation process step by step. This makes it particularly suitable for beginners or users who are not comfortable with command-line interfaces (CLI).

    2. **Ease of Use**: With the Python installer, you can simply download the installer package from the Python website and run it like any other application installer. This eliminates the need to use complex command-line commands, making the installation process more accessible to a wider range of users.

    ## via Homebrew

    In this instruction, we will install Python via `Homebrew`.
    If you don't have `Homebrew` installed on your macOS, please follow this instruction:

    <Card icon={<BrewIcon />} title="How to Install Homebrew on macOS" href="/docs/homebrew"></Card>

    <Callout >
    If you already have `Homebrew` installed on your macOS, you can skip this step.
    </Callout >

    <Steps>

    ### Step 1. Update Homebrew

    Before installing Python 3.10, ensure that Homebrew is up to date by running the following command in your terminal:

    ```bash copy filename="sh"
    brew update
    ```

    ### Step 2. Install Python 3.10

    Once Homebrew is updated, you can install Python 3.10 by executing the following command:

    ```bash copy filename="sh"
    brew install python@3.10
    ```

    This command will download and install Python 3.8 along with its dependencies.

    ### Step 3. Verify Installation

    After the installation is complete, you can verify that Python 3.10 has been successfully installed by checking its version:

    ```bash copy filename="console"
    $ python3 --version
    > Python 3.10.xx
    ```

    This output confirms that Python 3.10 has been installed successfully.

    ### Optional: Set Python 3.10 as Default (if needed)

    If you want to make Python 3.10 your default Python version, you can use the `pyenv` utility. First, install `pyenv` via Homebrew:

    ```bash copy filename="sh"
    brew install pyenv
    ```

    Then, set Python 3.10 as the global version:

    ```bash copy filename="sh"
    pyenv global 3.10
    ```
    </Steps>

        <Steps>
        ### Step 1: Miniconda Installation

        Commence by installing Miniconda through Homebrew:

        ```bash copy filename="sh"
        brew install --cask miniconda
        ```

        ### Step 2: Configuring Miniconda

        Following installation, configure Miniconda for your shell environment by executing the subsequent command:

        ```bash
        conda init "$(basename "${SHELL}")"
        ```

        Alternatively, you can manually append the following line to your shell initialization file (e.g., `~/.zshrc`, `~/.bashrc`):

        ```bash copy showLineNumbers filename="~/.zshrc"
        # Integrate Miniconda initialization into shell configuration
        eval "$(conda "shell.$(basename "${SHELL}")" hook)"
        ```
        or
        ```bash copy filename="sh"
        echo 'eval "$(conda "shell.$(basename "${SHELL}")" hook)"' >> ~/.zshrc
        ```
        </Steps>

        <Steps>
        ### Step 1: Install Anaconda

        - Use Homebrew to install Anaconda by running the following command:

          ```bash copy filename="sh"
          brew install --cask anaconda
          ```

        ### Step 2: Initialize Anaconda

        - After Anaconda is installed, open a new terminal window.

        - Run the following command to initialize Anaconda:

          ```bash copy filename="sh"
          conda init
          ```

        - Close the terminal window and open a new one to activate the changes.

        ### Step 3: Create a new environment (optional but recommended)

        - It's recommended to create separate environments for different projects to manage dependencies.

        - To create a new environment with Python 3.10, execute the following command:

          ```bash copy filename="sh"
          conda create --name myenv python=3.10
          ```
          Replace `myenv` with your preferred environment name.

        ### Step 4: Activate the environment

        - Activate the environment by running:

          ```bash copy filename="sh"
            conda activate myenv
            ```
            Replace `myenv` with the name of your environment.

        ### Step 5: Verify Python installation

        - To verify that Python 3.10 is installed, run:

            ```bash copy filename="sh"
            python --version
            ```

        - You should see output similar to:

            ```bash filename="console"
            $ python --version
            > Python 3.10.14
            ```

        </Steps>
