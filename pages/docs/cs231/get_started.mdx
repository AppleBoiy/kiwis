## Prerequisites

For this guide, you will need to have the following tools installed on your system:

- **MASM**: Microsoft Macro Assembler (MASM) is an x86 assembler.
- **DOSBox**: DOSBox is an x86 emulator that allows you to run old DOS applications on modern operating systems.
